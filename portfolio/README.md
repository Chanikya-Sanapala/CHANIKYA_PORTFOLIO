- Dark theme with subtle gradients

- Project cards with tech badges

## Quickstart

Install dependencies:

```bash
npm install
```

Run dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
npm run start
```

## Deploy
Deploy to Vercel or any static Node host supporting Next.js. Ensure `NODE_ENV=production` for production builds.
